# 微软如何虐待其用户

> 原文：[How Microsoft abuses its users](https://lzon.ca/posts/other/microsoft-user-abuse/)  
> 作者：lzon.ca  
> 翻译时间：2026-04-10

我想讲述一下我刚刚为一位客户完成的工作故事，这样我就可以阐明我对微软和其他大型科技公司如何对其用户采取主动敌对态度的看法。

## 故事背景

我接到邻居的电话，询问我是否愿意帮助她的丈夫解决他笔记本电脑上的问题。作为自己 IT 服务公司的骄傲新老板，我当然同意帮忙。

我与邻居的丈夫交谈后，立刻发现他不具备技术素养。我在为前雇主做 IT 工作时学会了识别这类人群。这使我理解他的问题变得困难，但通过对话，我们确实设法达成了对他所经历的真实问题的共识。

他看到的是他不再在 Outlook 中收到电子邮件，并且有一条错误消息声称他已经"耗尽了可用存储空间"，或者其他类似的废话。他是一个非常轻度的电子邮件用户，他知道这一点。他对为什么会耗尽存储空间感到困惑。起初我也很困惑。

通过调查，我发现 Outlook 电子邮件服务使用 Onedrive 来存储所有邮件和附件。他有 5 GB 的可用存储空间，这是他的免费账户所提供的数量。但这仍然无法解释为什么他会看到那条错误消息，他不可能仅凭电子邮件使用就消耗了 5 GB 的存储空间。

不出所料，他的 Onedrive 存储空间并不是被他的电子邮件填满的，而是被他的 Windows 11 桌面上的个人文件填满的。是他配置 Windows 将这些文件保存到他的 Onedrive 目录而不是本地主目录吗？当然不是，这是默认完成的。他甚至知道这正在发生吗？同样，不知道。他完全不知道这正在发生，直到他看到那条错误消息，该消息"如此贴心"地为他提供了解决方案——通过订阅该账户上的额外付费存储容量。

他确实设法大致理解了正在发生的事情，至少足以开始从他的计算机中删除文件以试图让错误消息消失。我永远无法与他确认，但我怀疑他删除了一些文件（包括家庭照片），而这些文件他没有其他备份。

我会直言不讳，这让我非常愤怒。这不是我第一次看到这种情况。我在为前雇主工作时曾多次看到这种情况。微软有意破坏了关于文件如何在运行 Windows 的计算机上存储的基本假设。他们在没有询问用户的情况下这样做，并且没有充分解释他们做了什么。微软显然正在使用黑暗模式来迫使其用户为 Onedrive 存储付费。

我是一个计算机极客，如果你正在阅读这篇文章，你可能也是。我们可以毫不费力地自己更改那个设置，而且我们可能有重要数据的备份以防需要恢复。我会告诉你，许多人对他们的计算机使用极其功利。他们只在必须服务于生活中其他兴趣的程度内使用计算机。他们也相信他们的财产——那个花费了他们数百美元的设备——不会像一些后巷骗子那样试图欺骗他们。

这不是游戏。我的客户不是电子表格上的一个数字，仅仅是达到某个无用的 KPI 的增量。他删除了家庭照片以试图让那条错误消息消失，这样他就可以再次接收电子邮件。他可能不明白发生了什么，但他并不愚蠢。他怀疑这是为了让他为他不需要的东西付费的骗局，他只是不明白这个骗局是如何运作的。

## 解决方案

首先，我对他的数据进行了完整备份。我提取了我在机器上本地能找到的所有内容，以及 Onedrive 账户中的所有内容，包括回收站。内容不多，只有几 GB，我将其传输到了一个单独的 USB 驱动器上。

我小心地将所有文件从 Onedrive 目录结构中转移回他的主文件夹。Windows 文件资源管理器并没有让这个过程变得简单或直观。

我继续通过 Web 界面删除 Onedrive 账户中的所有内容。我确实注意到删除文件只是将它们移动到回收站，而这仍然被计入总存储使用量。我假设这是另一个微妙的黑暗模式。

我提到了更改设置作为解决这个问题的一种方法。我们在前雇主那里经常采取的方法是简单地在 Windows 启动列表中禁用 Onedrive。这在这种情况下可能有效，但我有一个更好的主意。完全移除 Onedrive。

我现在对如何做到这一点已经形成了肌肉记忆，如果你想知道的话，这是我使用的程序：

打开管理员终端并加载 [Chris Titus 的 winutil](https://github.com/christitustech/winutil)。

```
irm christitus.com/win | iex
```

- 打开 Tweaks 标签页
- 在 Advanced Tweaks 部分，选择"Remove Onedrive"
- 按下"Run Tweaks"按钮

这完全从 Windows 中移除了 Onedrive 应用程序，包括所有与其他程序的集成，例如文件资源管理器。

然后我继续删除 Onedrive 账户中的所有内容，包括回收站。错误消息终于在 Outlook 中消失了，他能够再次接收电子邮件消息。

## 要点

我可能是在向唱诗班布道，但无论如何我想用这篇文章作为我以自己的方式表达这些观点的机会。

- 微软对其用户是主动敌对的。
- 他们已经变成了一个组织混乱的组织，在那里追逐无关紧要的 KPI 已经变得比产品质量，甚至对用户的基本尊重更重要。
- 对其他所有大型消费科技公司，这句话也在不同程度上适用。

我认为这是糟糕的激励结构的结果。一种有毒的博弈论，多年来在没有适当审查的情况下得以展开。我内心的左派可能会认为这是[晚期资本主义](https://en.wikipedia.org/wiki/Late_capitalism)的表现。如果是这样，那么感觉我们距离午夜还有大约 30 秒。

我经常思考调整这些激励结构的可能方法，建立一个可以防止导致这一过程的第一步的选择架构。

像今天这样的日子，当我在思考这种胡扯如何影响真实的人时，我无法忽视这个循环中的人。人们需要真正为他们的选择负责，而不是仅仅在电子表格中的数字看起来正确时关闭他们的大脑。
