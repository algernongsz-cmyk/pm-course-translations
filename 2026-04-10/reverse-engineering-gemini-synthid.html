针对Google DeepMind SynthID水印系统的逆向工程项目。包含V1/V2/V3绕过工具和多分辨率光谱码本（Spectral Codebook），用于从Gemini生成的图像中提取和移除水印。

功能特点

  • SpectralCodebook — 多分辨率码本（1024×1024和1536×2816）
  • SynthIDBypass — V1/V2/V3三代绕过算法
  • RobustSynthIDExtractor — 多尺度水印检测器（90%准确率）
  • CLI工具 — 用于码本构建、绕过和检测
  • 完整分析 — FFT可视化、相位分析、载波发现

快速开始

1. 构建V3码本（多分辨率）

从命令行：

python src/extraction/synthid_bypass.py build-codebook \
    --black gemini_black \
    --white gemini_white \
    --watermarked gemini_random \
    --output artifacts/spectral_codebook_v3.npz

或从Python：

from src.extraction.synthid_bypass import SpectralCodebook

codebook = SpectralCodebook()

# 方法1：从黑/白参考图像（1024x1024）
codebook.extract_from_references(
    black_dir='gemini_black',
    white_dir='gemini_white',
)

# 方法2：从带水印的内容图像（1536x2816）
codebook.build_from_watermarked('gemini_random')

codebook.save('artifacts/spectral_codebook_v3.npz')
# 保存的配置文件：[1024x1024, 1536x2816]

2. 运行V3绕过（任意分辨率）

from src.extraction.synthid_bypass import SynthIDBypass, SpectralCodebook

codebook = SpectralCodebook()
codebook.load('artifacts/spectral_codebook_v3.npz')

bypass = SynthIDBypass()
result = bypass.bypass_v3(image_rgb, codebook, strength='aggressive')

print(f"PSNR: {result.psnr:.1f} dB")
print(f"使用的配置: {result.details['profile_resolution']}")
print(f"精确匹配: {result.details['exact_match']}")

从命令行：

python src/extraction/synthid_bypass.py bypass input.png output.png \
    --codebook artifacts/spectral_codebook_v3.npz \
    --strength aggressive

强度级别：gentle（最小，~45 dB）> moderate > aggressive（推荐）> maximum

3. 检测水印

python src/extraction/robust_extractor.py detect image.png \
    --codebook artifacts/codebook/robust_codebook.pkl

技术深度解析

SynthID工作原理（逆向工程）

┌──────────────────────────────────────────────────────────────┐
│                  SynthID编码器（在Gemini中）                    │
├──────────────────────────────────────────────────────────────┤
│  1. 选择与分辨率相关的载波频率                                   │
│  2. 为每个载波分配固定相位值                                    │
│  3. 神经编码器向图像添加学习到的噪声模式                         │
│  4. 水印不可见——分布在频谱中                                   │
├──────────────────────────────────────────────────────────────┤
│                  SynthID解码器（在Google中）                    │
├──────────────────────────────────────────────────────────────┤
│  1. 提取噪声残差（小波去噪）                                    │
│  2. FFT → 在已知载波频率处检查相位                              │
│  3. 如果相位与期望值匹配 → 已添加水印                           │
└──────────────────────────────────────────────────────────────┘

多分辨率光谱码本

码本捕获每个可用分辨率的水印配置：

  • 1024×1024配置：来自100张黑色 + 100张白色纯色Gemini输出
    • 黑色图像：水印几乎是整个像素内容
    • 白色图像（反转）：通过交叉验证确认载波
    • 黑/白一致性（|cos(phase_diff)|）过滤掉生成偏差
  • 1536×2816配置：来自88张多样化的带水印内容图像
    • 内容在图像间平均化；固定水印在相位一致性中保留
    • 水印幅度估计为avg_mag × coherence²

V3减法策略

绕过使用直接已知信号减法（不是维纳滤波器）：

  1. 置信度 = phase_consistency × cross_validation_agreement
  2. DC排除 — 软斜坡抑制低频生成偏差
  3. 每bin减法 = wm_magnitude × confidence × removal_fraction × channel_weight
  4. 安全上限 — 减法从不超过任何bin中图像能量的90-95%
  5. 多轮 — 递减强度计划（aggressive → moderate → gentle）捕获残余能量

核心模块

synthid_bypass.py

SpectralCodebook — 多分辨率水印指纹：

codebook = SpectralCodebook()
codebook.extract_from_references('gemini_black', 'gemini_white')  # 添加1024x1024配置
codebook.build_from_watermarked('gemini_random')                  # 添加1536x2816配置
codebook.save('codebook.npz')

# 之后：
codebook.load('codebook.npz')
profile, res, exact = codebook.get_profile(1536, 2816)  # 自动选择

SynthIDBypass — 三代绕过：

bypass = SynthIDBypass()

result = bypass.bypass_simple(image, jpeg_quality=50)           # V1
result = bypass.bypass_v2(image, strength='aggressive')          # V2
result = bypass.bypass_v3(image, codebook, strength='aggressive') # V3（最佳）

robust_extractor.py

多尺度水印检测器（90%准确率）：

from robust_extractor import RobustSynthIDExtractor

extractor = RobustSynthIDExtractor()
extractor.load_codebook('artifacts/codebook/robust_codebook.pkl')
result = extractor.detect_array(image)
print(f"已加水印: {result.is_watermarked}, 置信度: {result.confidence:.4f}")
