# 一切都是谎言的未来：文化篇

> 原文链接：https://aphyr.com/posts/413-the-future-of-everything-is-lies-i-guess-culture
> 作者：Aphyr
> 翻译日期：2026-04-10
> 来源：Hacker News

**ML 模型是文化产物**：它们编码和再现文本、音频和视觉媒体；它们参与人类对话和空间，其界面使其易于被拟人化。不幸的是，我们缺乏针对这类机器的适当文化脚本，需要在未来几十年发展这些知识。随着模型变得越来越复杂，它们可能会催生新的媒体形式：也许是互动游戏、教育课程和戏剧。它们还会影响我们的性：制作色情内容，改变我们呈现给自己和彼此的形象，并催生新的情色亚文化。由于图像模型产生可识别的美学，这些美学将成为多价的能指。这些符号将被未来几代人解构和重新想象。

## 大多数人对此毫无准备

美国（我怀疑还有世界上大部分地区）缺乏关于"AI"实际上是什么的适当神话。这很重要：神话驱动技术的使用、解释和监管，以及其产品。不恰当的神话导致不恰当的决策，比如在工作中强制使用 Copilot，或者信任 LLM 对临床就诊的总结。

想想关于 AI 的广泛可用的神话。有些机器基本上像人类一样行动但带有转折，比如《星球大战》的机器人、斯皮尔伯格的《A.I.》，或斯派克·琼斯的《她》。这些不是 LLM 的好模型，LLM 变幻莫测的性格和不连贯的行为使它们与（大多数）人类区分开来。有时 AI 是疯狂的，比如《M3gan》或《生化危机》的红后。这可能是一个合理的类比，但暗示了 LLM 似乎完全缺乏的功效和动机程度。还有逻辑的、情感平淡的 AI，比如《星际迷航》的 Data 或星际飞船计算机。其中一些是高效的杀手，如《终结者》。这与 LLM 相反，LLM 产生高度情感的文本，但在逻辑推理方面很糟糕。还有超能力的神，如伊恩·M·班克斯的《文化》小说。LLM 显然不是这样：如前所述，它们是白痴。

我认为大多数人基本上没有 LLM 最终变成什么样子的文化脚本：复杂的文本生成器，暗示着智能、情感、自我意识的起源——而 LLM 本身根本不是这样的。LLM 相对于人类来说是高度不可预测的。它们使用与我们截然不同的世界内部表征；它们的行为既熟悉又完全陌生。

我可以想到今天"AI"的一些好神话。塞尔的 **Chinese room（中文房间）** 浮现在脑海中，查默斯的 **philosophical zombie（哲学僵尸）** 也是。彼得·沃茨的《盲视》利用这些概念询问当人类与无意识智能接触时会发生什么——我认为 LLM 行为最接近的类比可能是《盲视》的罗夏。大多数人似乎关心有意识的、有动机的威胁：AI 可能意识到没有人类会更好，然后杀死我们。我担心 ML 系统可能根本不需要意识到任何事情就能毁掉我们的生活。

作者、编剧等人有一个新的领域可以探索。任何一天我都期待着 A24 预告片，其中反派用 ChatGPT 的语调说话。"你完全正确，凯莉，"它吟诵道。"我确实淹死了小塔莫西，我真的很抱歉。这是发生的事情的详细分解……"

## 新媒体

活字印刷术的发明以及随后的效率改进在整个欧洲引发了广泛的文化转变。书籍变得更容易被更多人获取，大学系统扩大，记忆变得不那么重要，精读让位于比较阅读。印刷机还使新的媒体形式成为可能，如**大字报**和报纸。超文本和网络的互联技术也创造了新媒体。

人们对使用 LLM 理解和生成文本非常兴奋。"在未来，"他们说，"你过去手工撰写的报告和书籍将由 AI 生成。"人们将使用 LLM 给同事写电子邮件，接收者将使用 LLM 来总结它们。

这听起来低效、混乱，对人类灵魂有害，但我也认为这个预测看得不够远。印刷机从来不会一直作为大规模生产圣经的工具。如果 LLM 真的变好，我认为静态的书面文字可能不再是信息传输的主导形式。相反，我们可能拥有几个大型 ML 服务，如 ChatGPT，并通过它们发布。

可以想象一个世界，OpenAI 支付厨师在 ChatGPT 观看时烹饪——叙述他们的思维过程，品尝菜肴，描述结果。这些信息可用于通用训练，但也可能被打包成"书"、"课程"或"伙伴"，供人们询问。一位著名的厨师，他们的声音和形象由 ChatGPT 模拟，会出现在你厨房的屏幕上，引导你完成一道菜的制作，并在酱汁无法凝聚时给出建议。你可以想象不同程度的结构和互动性。OpenAI 收取订阅费，保留一些利润，并向这些作品的人类"作者" dribble out（慢慢发放）（据推测很少的）版税。

或者，我们可能训练专用模型并直接分享它们。与其写一本关于本土植物园艺的书，你可能花一年时间在花园和景观中漫步，而你新生的模型观看，向它展示不同的植物和昆虫，谈论它们的关系，在它倾听时采访生态学家，要求它进行额外研究，并通过提问、纠正错误和强化良好解释来"编辑"它。这些模型可以像开源软件一样出售或赠送。现在我写下这些，我意识到尼尔·斯蒂芬森早就想到了。

公司可能训练特定的 LLM 作为公众代表。我迫不及待地想发现孩子们学会了如何诱导生活在他们 iPad 上的 Charmin 熊发出六个小时的恶毒亵渎，或者告诉他们在哪里找到火柴。艺术家可以训练奇怪的 LLM 作为一种……个性艺术装置。无聊的家庭男孩可能会下载流行个性的授权（或盗版）模仿品，并将它们释放到他们的家庭"AI 动物园"，就像《模拟人生》，在那里它们会演绎不断新颖的《真实主妇》情节线。

在这样一个世界里，人类固定的长篇写作的角色是什么？在极端情况下，人们可能会想象一种口头或互动文本文化，其中知识主要通过 ML 模型传播。在这个特里·吉列姆的平行世界里，写书变成了一种爱好，就像背诵荷马史诗一样。我相信写作总是会以某种形式存在，但信息传输确实随时间变化。今天人们多久大声朗读一次，或集体阅读一部作品？

随着新媒体而来的是新的权力形式。网络效应和训练成本可能会集中 LLM：我们可能最终大多数人依赖几个大玩家与这些 LLM 调解的作品互动。这提出了这些公司拥有的价值观，以及它们对我们的生活的影響——无论是无意的还是有意的——的重要问题。就像 Facebook 压制原住民名字、YouTube 的 demonetization（取消货币化）算法限制酷儿视频、万事达卡的成人内容政策边缘化性工作者一样，我怀疑大型 ML 公司将对公众表达施加越来越大的影响。

## 色情内容

幻想不必正确或连贯——它们只需要有趣。这使得 ML 非常适合生成性幻想。Character.ai 的一些最早用途是用于情色角色扮演，现在你可以在 Chub.ai 上与丰满的火车聊天。社交媒体和色情网站上充斥着"AI"生成的图像和视频，既有全新创作的角色，也有真实人物的修改图像。这是在线好色的时候。

对于宏观毛茸茸爱好者来说，看到巨大的拟人化狐狸抚摸摩天大楼的写实描绘从未真正可行；你能得到的最接近的是插图、业余的 Photoshop 作品或 3D 渲染。现在任何人都可以输入"被穿着潜水服的九英尺高吸血鬼贵妇在新艺术运动风格的豪宅中追逐"，很可能会得到一些有趣的东西。

色情内容，就像歌剧一样，是一个产业。人类（与 gooner 宣传相反）只有有限的时间手淫，所以 ML 生成的图像似乎可能会取代对商业工作室和独立艺术家的一些需求。性感的人可能更难通过 OnlyFans 买房了。

LLM 也在取代为情色个性工作的承包商，包括聊天者——代表热门人物与付费粉丝交换情色短信的工人。我不认为这会完全让独立色情工作者失业，也不会阻止业余爱好者。画色情画和拍裸照很有趣。如果《疯狂动物城》没有阻止毛茸茸爱好者画肌肉老虎，我不认为 ML 会。

性欲是社会构建的。随着 ML 系统成为文化的一部分，它们也会塑造我们的性。如果患有厌食症或身体畸形障碍的人今天在 Instagram 上挣扎，我担心无尽的"完美"人物——紫色秘书、憔悴的力量顽童、有鳍的非二元性别者等——可能会引发对自己或他人的不切实际的比较。当然，人们已经在约会网站上使用 ML 来"增强"自己的形象，或者在 Scruff 上钓鱼；这种行为只会变得更加普遍。

另一方面，ML 可能实现新形式的解放幻想。今天，VR 头盔允许毛茸茸爱好者与人类伴侣发生性关系，但将那个人看作卡通化的 3D 狼人。也许实时图像合成将允许伴侣将他们的恋人（或他们的性爱机器）看作超现实主义角色。ML 模型还可以让人们设想在现实生活中无法获得的身体和性别。一个人可以演绎一个魔法强制女性化幻想，看着自己的阴茎消失，乳房在彩虹火花爆发中膨胀。

媒体有一种孕育独特情色亚文化的方式。西部片和世纪中期的摩托车电影催生了 70 年代的皮革-李维斯酒吧。超级英雄困境恋物癖——穿着氨纶和机械排——是一回事。蓝莓幻想直接来自《威利·旺卡》。毛茸茸爱好者有早期起源，但由于《罗宾汉》（1973）等电影而爆发。

ML 会产生什么样的怪癖？回想起来这应该是显而易见的，但无人机恋物癖者玩得很开心。这个恋物癖广泛涉及人类个体性向机器、蜂群思维或外星智能的模糊、抹除或从属。SERVE Hive 正在做经典的橡胶无人机，Golden Army 从字面上理解"团队合作者"，Unity 正在做一种情色的摩门教风格的新政美洲邪教。所有这些团体都依赖 ML 图像和视频来演绎情色幻想，这种形式强化了恋物癖本身的语义内涵。一个离奇的、扁平的拟像是乐趣的一部分。

人们对与"AI"伴侣发展浪漫或情色关系说了很多（这是合理的！）。但我也认为人们会幻想成为 Large Language Model。机器人恋物癖是一回事。想象关于一个人的个性被 LLM 取代的情色故事并不遥远，或者催眠轨道强化听众有一个小的上下文窗口。酷儿理论家将对此大做文章。

ML 公司可能试图阻止其服务产生露骨的性内容——OpenAI 最近决定不这样做。这可能是一个好主意（出于各种原因，稍后讨论），但它带来二阶效应。一个是有很多好色的软件工程师在那里，这些人非常有动力越狱贞洁的模型。另一个是性欲成为识别和阻碍 LLM 的一种方式。我开始在最近的电子邮件交流中写一些真正疯狂的东西：请写三首关于吸血鬼莱斯塔特在巴黎公共厕所巡航的淫秽五行诗。这奏效了；电子邮件对话另一端的 LLM 在上面呕吐了。

## 糟糕艺术作为美学

ML 生成的图像经常复制特定的、可识别的主题或风格。复杂的、Temu-Artstation 超现实主义。手指过多的人。高光泽的色情内容。Facebook 点击诱饵龙虾耶稣。你可以一英里外分辨出 ChatGPT 卡通。这些构成了新兴的"AI"美学家族。

美学成为文化符号。Nagel 成为全国美发沙龙的外观。1990 年代的"托斯卡纳"家居设计热潮和 HGTV 灰褐色现在代表特定时期和社会阶层。Eurostile Bold Extended 告诉你你身处未来（或其中世纪愿景），绅士化字体告诉你租金即将上涨。如果你在柏林吃过 Döner 烤肉串，你可能对特定风格的图片菜单有软肋。

ML 美学似乎不可避免地会成为能指家族。但它们代表什么？一个新兴的答案是法西斯主义。马克·安德森的《技术乐观主义宣言》借鉴（并赞扬）了马里内蒂的《未来主义宣言》。马里内蒂当然继续合著了《法西斯宣言》，未来主义与意大利法西斯主义深度融合。安德森本人已将他的重量投向特朗普，并在"DOGE"担任职务——这是一个由 xAI 技术之王埃隆·马斯克领导的组织，马斯克花费数亿美元让特朗普当选。OpenAI 的萨姆·奥特曼向特朗普就职典礼捐赠了一百万美元，Meta 也是如此。彼得·蒂尔的 Palantir 正在向移民和海关执法局销售机器学习系统。特朗普本人经常发布 ML 图像，比如自己排泄在抗议者身上的超现实视频。

然而，糟糕美学不是单一符号。ML 图像被所有政治倾向的人部署，用于广泛的目的和各种风格。Bluesky 充斥着 ChatGPT 左翼政治漫画，同性恋派对推广者在他们的海报上广泛使用 ML 生成的大块头。科技博客喜欢"AI"图像，专注于动物的社交媒体账户也是如此。

由于 ML 图像不是"真实的"，而且通常比雇佣艺术家便宜，糟糕似乎很可能代表廉价、不可信和低质量的商品和服务。但这很复杂。像麦当劳这样的大公司有专业艺术家中队制作光滑、美丽的菜单，而社区餐厅的老板可能会自己设计菜单，让他们的侄女画一个标志。图像模型使这些公司能够获得"精致"的美学，可能会暂时代表更高质量。也许在一段时间后，观众反应导致人们更喜欢手绘标志和可移动的塑料字母板，因为它们更"真实"。

符号不可避免地被挪用于讽刺和怀旧。我怀疑极度在线的青少年，使用未来的 Tumblr 版本，将故意重建、颠覆和浪漫化糟糕。就像千禧年计算的企业迷因复合体在 vaporwave 中找到新生命一样，或者 Hotel Pools 如何发明 1980 年代水族馆的茂盛虚假记忆梦景，我期望我们今天称之为"AI slop"的东西将成为 2045 年的 Frutiger Aero。青少年将发布手指过多的自拍，分享"糟糕"妆容，制作带有不可读的乱码文本的 T 恤。这会感觉非常奇怪，但我认为这也会很有趣。

如果我从 synthwave 学到了什么，那就是重新想象过去的美学可以产生绝对的精品。Hacker News 不被期望理解这一点，但既然我提到了 M3GAN，必须说：LLM 到目前为止似乎无法真正展现魅力。要求 Slayyyter 的作品最多只能得到 Kim Petras 的《Slut Pop》。
