Project Glasswing：为 AI 时代保护关键软件

原文：https://www.anthropic.com/glasswing

引言

今天我们宣布 Project Glasswing1，这是一项新的倡议，联合了 Amazon Web Services、Anthropic、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorganChase、Linux Foundation、Microsoft、NVIDIA 和 Palo Alto Networks，致力于保护世界上最关键的软件。

我们发起 Project Glasswing 是因为我们在 Anthropic 训练的一个新前沿模型中观察到了某些能力，这些能力可能会重塑网络安全。Claude Mythos2 Preview 是一个通用的、未发布的前沿模型，它揭示了一个严峻的事实：AI 模型在编码能力上已经达到了一个水平，可以在发现和利用软件漏洞方面超越除最熟练的人类之外的所有人。

Mythos Preview 已经发现了数千个高危漏洞，包括在每一个主要操作系统和每一个主要网页浏览器中的漏洞。考虑到 AI 的进步速度，这些能力不久之后就会扩散，可能超出那些致力于安全部署这些能力的参与者。对经济、公共安全和国家安全的影响可能是严重的。Project Glasswing 是一个紧急尝试，旨在将这些能力用于防御目的。

作为 Project Glasswing 的一部分，上述启动合作伙伴将使用 Mythos Preview 作为其防御性安全工作的一部分；Anthropic 将分享我们的学习成果，以便整个行业都能受益。我们还将访问权限扩展到另外 40 多个构建或维护关键软件基础设施的组织，以便它们可以使用该模型扫描和保护自有系统和开源系统。Anthropic 承诺为这些工作提供高达 1 亿美元的 Mythos Preview 使用额度，以及向开源安全组织直接捐赠 400 万美元。

Project Glasswing 是一个起点。没有任何一个组织能够独自解决这些网络安全问题：前沿 AI 开发者、其他软件公司、安全研究人员、开源维护者和全球各国政府都扮演着重要角色。防御世界网络基础设施的工作可能需要数年时间；而前沿 AI 能力可能在未来几个月内就会大幅进步。为了让网络防御者占据优势，我们需要现在就行动。

AI 时代的网络安全

我们每天都在依赖的软件——负责运行银行系统、存储医疗记录、连接物流网络、维持电网运行等等——一直以来都包含漏洞。许多漏洞是次要的，但有些是严重的安全缺陷，如果被发现，可能允许网络攻击者劫持系统、破坏操作或窃取数据。

我们已经看到了网络攻击对重要的企业网络、医疗系统、能源基础设施、交通枢纽和全球政府机构信息安全的严重后果。在全球舞台上，来自中国、伊朗、朝鲜和俄罗斯等国家支持的攻击者威胁着破坏支撑平民生活和军事准备的基础设施。即使是小规模的攻击，例如针对单个医院或学校的攻击，仍然可能造成巨大的经济损失、暴露敏感数据，甚至危及生命。目前全球网络犯罪的财务成本难以估算，但可能每年约为 5000 亿美元。

许多软件缺陷多年来未被注意，因为发现和利用它们需要少数熟练安全专家才具备的专业知识。随着最新的前沿 AI 模型，发现和利用软件漏洞所需的成本、努力和专业水平都大幅下降。在过去一年中，AI 模型在阅读和推理代码方面变得越来越有效——特别是，它们显示出发现漏洞和开发利用方式的惊人能力。Claude Mythos Preview 在这些网络技能上展现了飞跃——它发现的漏洞在某些情况下经历了数十年的人工审查和数百万次自动化安全测试而未被发现，它开发的漏洞利用也越来越复杂。

在首次 DARPA 网络大挑战赛十年后，前沿 AI 模型现在在发现和利用漏洞方面正在与最优秀的人类竞争。如果没有必要的保障措施，这些强大的网络能力可能会被用来利用世界上最重要软件中的许多现有缺陷。这可能使各种网络攻击变得更加频繁和具有破坏性，并增强美国及其盟友的对手的能力。因此，解决这些问题是民主国家的重要安全优先事项。

虽然 AI 增强的网络攻击风险是严重的，但也有理由乐观：使 AI 模型在错误的手中变得危险的相同能力，使它们在发现和修复重要软件缺陷方面具有不可估量的价值——并且可以生成安全漏洞少得多的新软件。Project Glasswing 是让防御者在即将到来的 AI 驱动的网络安全时代获得持久优势的重要一步。

使用 Claude Mythos Preview 识别漏洞和利用方式

在过去几周内，我们使用 Claude Mythos Preview 识别了数千个零日漏洞（即软件开发者以前不知道的缺陷），其中许多是关键漏洞，涉及每一个主要操作系统和每一个主要网页浏览器，以及其他重要的软件。

在我们的前沿红队博客上的一篇文章中，我们提供了已经修补的这些漏洞的一个子集的技术细节，以及 Mythos Preview 发现的利用它们的方式。它能够几乎完全自主地识别几乎所有这些漏洞——并开发许多相关的利用方式——无需任何人工引导。以下是三个例子：

  • Mythos Preview 在 OpenBSD 中发现了一个 27 年前的漏洞——OpenBSD 作为世界上最安全的操作系统之一而闻名，用于运行防火墙和其他关键基础设施。该漏洞允许攻击者仅通过连接到运行该操作系统的任何机器就能远程使其崩溃；
  • 它还在 FFmpeg 中发现了一个 16 年前的漏洞——FFmpeg 被无数软件用于编码和解码视频——在一行代码中，自动化测试工具已经命中了 500 万次，却从未发现问题；
  • 该模型自主发现并链接了 Linux 内核中的几个漏洞——Linux 内核是运行世界上大多数服务器的软件——允许攻击者从普通用户访问升级到对机器的完全控制。

我们已经向相关软件的维护者报告了上述漏洞，它们现在都已修补。对于许多其他漏洞，我们今天提供详细信息的加密哈希（见红队博客），我们将在修复到位后揭示具体细节。

我们不打算让 Claude Mythos Preview 普遍可用，但我们的最终目标是使我们的用户能够安全地在规模上部署 Mythos 级别的模型——用于网络安全目的，也用于此类高能力模型将带来的无数其他好处。为此，我们需要在开发网络安全（和其他）保障措施方面取得进展，这些保障措施可以检测和阻止模型最危险的输出。我们计划在即将推出的 Claude Opus 模型中推出新的保障措施，使我们能够改进和完善它们，使用的模型不会像 Mythos Preview 那样构成相同级别的风险3。

Project Glasswing 的计划

今天的宣布是一个更长期努力的开端。要取得成功，将需要整个技术行业及更广泛领域的广泛参与。

Project Glasswing 合作伙伴将获得 Claude Mythos Preview 的访问权限，以发现和修复其基础系统中的漏洞或弱点——这些系统代表了世界上共享网络攻击面的很大一部分。我们预计这项工作将专注于本地漏洞检测、二进制黑盒测试、保护端点和系统渗透测试等任务。

Anthropic 承诺为 Project Glasswing 提供 1 亿美元的模型使用额度和额外参与者，将覆盖整个研究预览期间的实质性使用。之后，Claude Mythos Preview 将以每百万输入/输出令牌 25 美元/125 美元的价格提供给参与者（参与者可以在 Claude API、Amazon Bedrock、Google Cloud 的 Vertex AI 和 Microsoft Foundry 上访问该模型）。

除了我们的模型使用额度承诺外，我们通过 Linux Foundation 向 Alpha-Omega 和 OpenSSF 捐赠了 250 万美元，向 Apache Software Foundation 捐赠了 150 万美元，使开源软件的维护者能够应对这个不断变化的格局（感兴趣的维护者可以通过 Claude for Open Source 项目申请访问权限）。

我们打算让这项工作的范围扩大并持续数月，我们将尽可能多地分享，以便其他组织可以将这些经验应用到自己的安全中。合作伙伴将在可能的范围内相互分享信息和最佳实践；在 90 天内，Anthropic 将公开发布我们学到的内容，以及可以披露的已修复漏洞和改进。我们还将与领先的安全组织合作，制定一套关于安全实践如何在 AI 时代演变的实用建议。这可能包括：

  • 漏洞披露流程；
  • 软件更新流程；
  • 开源和供应链安全；
  • 软件开发生命周期和安全设计实践；
  • 受监管行业的标准；
  • 分流扩展和自动化；以及
  • 补丁自动化。

Anthropic 还一直在与美国政府官员就 Claude Mythos Preview 及其攻击性和防御性网络能力进行持续讨论。正如我们上面指出的，保护关键基础设施是民主国家的首要国家安全优先事项——这些网络能力的出现是美国及其盟友必须保持 AI 技术决定性领先优势的另一个原因。各国政府在帮助保持这一领先优势以及评估和缓解与 AI 模型相关的国家安全风险方面发挥着重要作用。我们准备与地方、州和联邦代表合作，协助完成这些任务。

我们希望 Project Glasswing 能够在行业和公共部门之间播下更大努力的种子，各方都帮助解决围绕强大模型对安全影响的最大问题。我们邀请其他 AI 行业成员加入我们，帮助为行业设定标准。在中期，一个独立的第三方机构——可以汇集私营和公共部门组织——可能是继续开展这些大规模网络安全项目的理想场所。
