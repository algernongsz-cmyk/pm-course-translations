介绍 Sales Agent 蓝图

今天我们宣布推出 Sales Agent 蓝图，这是一份路线图，帮助你自信地启动和扩展销售 AI。

我们最初推出 Service Agent 蓝图，是因为许多支持团队对 AI 的前进路径不够清晰。Go-to-market 和营收团队现在也处于同样的境地，因此今天我们宣布 Sales Agent 蓝图——一份帮助营收团队应对销售转型关键时刻的战略路线图。

对大多数销售团队来说，inbound 销售流程需要大量手工工作。潜在客户进来后，会在队列中等待，直到有人有空处理。结果往往取决于销售团队的“速度致胜”能力——快速有效响应的能力。响应越慢，潜在客户转化的可能性就越低。

Agent 可以端到端地运行销售对话——与买家互动、筛选潜在客户，并将高意向的机会路由到合适的团队，快速推进潜在买家。人类仍然会完成交易，但 AI Agent 将确保他们专注于最佳机会和最优质、最合格的潜在客户。

许多 Go-to-market 和营收领导者现在面临的问题是：从哪里开始？成功应该是什么样的？如何实际测试和部署这些解决方案？

Sales Agent 蓝图回答了这些问题。它专为希望快速部署 inbound 销售 AI、证明价值并为长期转型奠定基础的销售、营收和 AI 转型领导者设计。

内容概览

Sales Agent 蓝图围绕两个轨道构建：

1. 启动它（现已上线）

今天，我们发布蓝图的第一部分：“启动它”。这是一份实用指南，帮助你让 Agent 上线并看到实际效果。学习如何部署一个能够运行 inbound 销售对话、筛选潜在客户并将机会路由到合适团队的 Sales Agent——7×24小时运行，规模化处理，无需人工干预。

完成“启动它”部分后，你将知道如何：

  • 理解什么是 Sales Agent：了解它们为什么与聊天机器人不同以及如何工作。
  • 建立商业案例：证明 AI 的基本经济效益，决定是购买还是自建，并获得所需预算。
  • 评估 Agent：学习如何定义成功、选择正确的评估标准，并使用我们的五步框架进行有针对性、高影响力的评估。
  • 自信部署：制定部署计划，让 Agent 快速上线，在买家意向高峰时互动。了解每个阶段的预期。
  • 持续改进性能：上线后，你的 Agent 成为一个需要管理的系统。我们将向你展示如何实施可重复的训练、测试、部署和优化流程。

2. 扩展它（即将推出）

启动 AI 只是开始。要释放其全部潜力，你需要重构你的 inbound 销售流程——重新设计买家旅程、构建 AI 优先的系统和所有权模型，并重新思考团队培训方式和成功衡量标准。

我们自己也在进行这一旅程，因此我们将在探索过程中公开构建蓝图。
