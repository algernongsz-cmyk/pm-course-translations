Twill 在隔离的云沙箱中运行 Claude Code 和 Codex 等编码 CLI 工具。你通过 Slack、GitHub、Linear 等平台将工作交给它。

核心功能

Twill 是一个云端的编码 Agent 平台，可以：

- 在隔离的云沙箱环境中运行编码 CLI 工具
- 支持 Claude Code、Codex 等主流编码工具
- 通过 Slack、GitHub、Linear 等常用平台接收任务
- 自动处理编码任务并返回 PR（Pull Request）

使用场景

Twill 适合需要自动化编码工作的团队：

- 开发团队可以通过 Slack 或 GitHub 直接分配编码任务
- Agent 会在云端沙箱中独立工作，不影响本地环境
- 完成后返回可直接合并的 PR
