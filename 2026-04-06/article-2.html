AI Edge Gallery：在移动设备上运行世界最强大开源 LLM

AI Edge Gallery 是在移动设备上运行世界最强大开源大型语言模型（LLM）的首选平台。直接在您的硬件上体验高性能生成式 AI——完全离线、私密且极速。

现已支持：Gemma 4

此更新带来了对新发布的 Gemma 4 系列的官方支持。作为此次更新的核心，Gemma 4 让您能够测试设备端 AI 的前沿技术。体验先进的推理、逻辑和创造能力，无需将数据发送到服务器。

核心功能

  • Agent Skills：将您的 LLM 从对话者转变为主动助手。使用 Agent Skills 磁贴来增强模型能力，配备维基百科事实支撑、交互式地图和丰富的可视化摘要卡等工具。您甚至可以从 URL 加载模块化技能，或在 GitHub Discussions 上浏览社区贡献。
  • AI Chat with Thinking Mode：进行流畅的多轮对话，并切换新的 Thinking Mode 来"查看内部运作"。此功能让您看到模型的逐步推理过程，非常适合理解复杂问题解决。注意：Thinking Mode 目前适用于支持的模型，从 Gemma 4 系列开始。
  • Ask Image：使用多模态能力来识别物体、解决视觉谜题，或使用设备相机或照片库获取详细描述。
  • Audio Scribe：使用高效的设备端语言模型实时转录和翻译语音录音。
  • Prompt Lab：一个专门的工作空间，用于测试不同的提示词和单轮用例，对温度和 top-k 等模型参数进行精细控制。
  • Mobile Actions：解锁离线设备控制和自动化任务，完全由 FuntionGemma 270m 的微调版本驱动。
  • Tiny Garden：一个有趣的实验性迷你游戏，使用自然语言种植和收获虚拟花园，由 FunctionGemma 270m 的微调版本驱动。
  • Model Management & Benchmark：Gallery 是各种开源模型的灵活沙盒。轻松从列表下载模型或加载您自己的自定义模型。轻松管理您的模型库，并运行基准测试以准确了解每个模型在您的特定硬件上的性能。
  • 100% 设备端隐私：所有模型推理直接在您的设备硬件上进行。不需要互联网，确保您的提示词、图像和敏感数据的完全隐私。

为社区而构建

AI Edge Gallery 是一个开源项目，专为开发者社区和 AI 爱好者设计。探索我们的示例功能，贡献您自己的技能，帮助塑造设备端 Agent 生态系统的未来。
