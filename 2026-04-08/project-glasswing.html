Project Glasswing：为AI时代保障关键软件安全

来源： Anthropic
翻译时间： 2026-04-08
类型： AI安全 / 网络安全
⚠️ 翻译说明：本文翻译自 Anthropic 官方网站。Project Glasswing 是 Anthropic 启动的一个安全项目，联合多家科技公司，使用 Claude Mythos Preview 模型发现和修复软件漏洞。

项目概述

今天我们宣布启动 Project Glasswing，这是一个新的安全倡议，联合了 Amazon Web Services、Anthropic、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorganChase、Linux Foundation、Microsoft、NVIDIA 和 Palo Alto Networks，旨在保障世界上最关键软件的安全。

AWS Apple Google Microsoft NVIDIA Cisco CrowdStrike Palo Alto Networks

我们发起 Project Glasswing 的原因，是因为我们观察到 Anthropic 训练的一个新前沿模型展现出的能力，可能会重塑网络安全格局。Claude Mythos Preview 是一个通用、尚未发布的前沿模型，它揭示了一个严峻的事实：AI 模型的编程能力已经达到了可以超越几乎所有人类专家的水平，能够发现和利用软件漏洞。

关键发现：Mythos Preview 已经发现了数千个高危漏洞，包括所有主要操作系统和网页浏览器中的漏洞。

AI时代的网络安全

我们每天依赖的软件——负责运行银行系统、存储医疗记录、连接物流网络、维持电网运行等等——一直都存在漏洞。许多是次要的，但一些是严重的安全缺陷，如果被发现，可能允许网络攻击者劫持系统、破坏运营或窃取数据。

我们已经看到网络攻击对重要的企业网络、医疗系统、能源基础设施、交通枢纽和全球政府机构信息安全的严重后果。在全球舞台上，来自中国、伊朗、朝鲜和俄罗斯等国家支持的行为者的攻击，威胁着民用生活和军事准备的基础设施。

Mythos Preview 的能力

在过去几周里，我们使用 Claude Mythos Preview 识别了数千个零日漏洞（即软件开发者之前未知的缺陷），其中许多是关键的，涉及每个主要操作系统和每个主要网页浏览器，以及一系列其他重要软件。

我们已经向相关软件的维护者报告了上述漏洞，现在它们都已修补。 对于许多其他漏洞，我们今天提供细节的加密哈希，并在修复到位后公布具体信息。

评估基准

CyberGym 等评估基准强调了 Mythos Preview 与我们次优模型 Claude Opus 4.6 之间的显著差异：

  • 在网络安全任务上，Mythos Preview 的表现大幅领先其他模型
  • 强大的代理式编程和推理能力是其网络能力的基础
  • 在各种软件编程任务上取得了迄今开发的任何模型中的最高分

项目承诺

作为 Project Glasswing 的一部分：

  • 1亿美元模型使用额度：Anthropic 承诺为 Project Glasswing 和其他参与者提供 1 亿美元的模型使用额度
  • 400万美元捐赠：直接捐赠给开源安全组织
  • 40+组织参与：超过 40 个额外组织已获得访问权限，用于扫描和保护其系统

定价与访问

Claude Mythos Preview 将以每百万输入/输出 token 25美元/125美元的价格提供给参与者（参与者可通过 Claude API、Amazon Bedrock、Google Cloud 的 Vertex AI 和 Microsoft Foundry 访问该模型）。

后续计划

我们打算让这项工作在范围上不断扩大，持续数月，并尽可能多地分享，以便其他组织可以将这些经验应用到自己的安全工作中。

  • 合作伙伴将尽可能相互分享信息和最佳实践
  • 90天内，Anthropic 将公开报告我们的经验教训
  • 将与领先的安全组织合作，制定 AI 时代安全实践的实用建议

可能包括的建议

  • 如何在 AI 时代重新设计安全审计流程
  • 如何平衡漏洞披露的速度与修复时间
  • 如何建立跨组织的漏洞共享机制

与政府的合作

Anthropic 一直在与美国政府官员就 Claude Mythos Preview 及其进攻性和防御性网络能力进行持续讨论。保护关键基础设施是民主国家的国家安全优先事项——这些网络能力的出现是美国及其盟友必须在 AI 技术上保持决定性领先优势的另一个原因。

未来展望

我们希望 Project Glasswing 能够成为跨行业和公共部门更大努力的种子，各方共同帮助解决围绕强大模型对安全影响的最大问题。我们邀请其他 AI 行业成员加入我们，共同为行业制定标准。

从中长期来看，一个独立的第三方机构——一个能够将私营和公共部门组织聚集在一起的机构——可能是继续开展这些大规模网络安全项目的理想场所。
